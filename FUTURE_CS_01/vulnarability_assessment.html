Assessment Methodology

How VulnScout conducts passive vulnerability assessments — aligned with OWASP WSTG and real-world security consulting practice.

✅ What's Allowed (Scope)

  • Public-facing pages only
  • Passive HTTP/HTTPS scanning
  • Security header inspection
  • Response body analysis
  • Cookie flag checks
  • TLS/HTTPS configuration review
  • Information disclosure detection
  • Publicly accessible resource enumeration

🚫 What's Not Allowed

  • Login bypass or credential attacks
  • SQL injection or exploitation
  • Brute force attacks
  • Denial of Service (DoS)
  • Accessing private APIs without permission
  • Testing systems you don't own
  • Automated aggressive scanning
  • Any action that modifies site data
Step-by-Step Process
1

Target Selection

Choose a public website you own, a demo site, or a site explicitly designed for testing (e.g. testphp.vulnweb.com). Never test sites without permission.

2

HTTP Request & Header Collection

Perform a GET request to the target. Collect all response headers including security headers, server info, cookies, and redirect behavior.

3

Security Header Analysis

Check for presence and correct configuration of: HSTS, CSP, X-Content-Type-Options, X-Frame-Options, Referrer-Policy, Permissions-Policy, and CORS headers.

4

Information Exposure Check

Inspect for server software disclosure (Server header, X-Powered-By), verbose error messages, internal paths, version numbers, and technology stack fingerprinting.

5

TLS / HTTPS Verification

Check if site uses HTTPS, if HTTP redirects to HTTPS, and if HSTS is enforced with a sufficient max-age directive.

6

Risk Classification & AI Analysis

Each finding is classified as Critical / High / Medium / Low / Info based on exploitability, impact, and CVSS-aligned scoring. AI provides plain-English business impact explanations.

7

Report Generation

All findings are compiled into a professional Vulnerability Assessment Report with executive summary, detailed findings, business impact, and prioritised remediation steps.
